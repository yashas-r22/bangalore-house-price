# 🏡 Bangalore House Price Predictor

A Machine Learning web app to predict house prices in Bangalore.

## 🔧 Tech Stack
- Python, Pandas, NumPy
- Scikit-learn (Linear Regression, Random Forest, Gradient Boosting)
- Streamlit (Web App)

## 📊 Dataset
Bengaluru House Price Data — Kaggle

## 🚀 How to Run
1. pip install -r requirements.txt
2. python model.py
3. streamlit run app.py

## 📈 Model Performance
- Best Model: Random Forest / Gradient Boosting
- R² Score: ~0.85+
- MAE: ~12 Lakhs
